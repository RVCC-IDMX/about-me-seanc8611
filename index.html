The topic I decided to research was CSS animation. CSS animation is a feature of CSS that allows you to animate a change in style properties of an element.

You can also control the different aspects of the animation. There is no need for additional code as it is done with just HTML and CSS.

The main way to use CSS animation is you need to specify some keyframes for the animation. Keyframes hold the styles the element will have.

You can bind the animation to an element, and the animation-duration property defines how long an animation will take.

There are certain advantages that CSS animations have over traditional script-driven animation. For simple animations, it is straight-forward and easy to use. It also doesn't use a moderate system load so the rendering engine can use frame-skipping to keep the performance smooth. Lastly, it lets the browser control the animation sequence to optimize performance and efficiency.
